In each field below, paste the full path to the resource. Click "Done" once complete.

To get the paths browse to the file/directory. For windows, hold SHIFT then right-click and select "copy as path". For MacOS right-click then hold OPTION and select "Copy as Pathname".

Example Windows paths: "C:\Dolphin\Dolphin.exe", "C:\Users\user\Games"

Example MacOS paths: /Applications/Dolphin.app, /Users/user/games
